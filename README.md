# Asia World Cup Resource

本项目用于归档和发布多个独立的 HTML 页面，内容主要围绕亚洲杯相关主题。所有页面均为静态 HTML 文件，不依赖后端服务或特定域名。

## 项目简介

本仓库旨在提供一份可公开访问的页面归档，便于浏览、分享或嵌入其他场景。页面内容独立，无跨站依赖，适合直接部署于各类静态托管环境。

## 目录说明

```
.
├── index.html          # 入口页面，提供导航或简介
├── pages/              # 存放主要页面文件（HTML）
│   └── ...
├── assets/             # 静态资源（CSS、JS、图片等）
│   └── ...
└── README.md           # 本文件
```

- `index.html`：仓库根目录下的主页面，可作为访问入口。
- `pages/`：按主题或时间归档的页面文件，每个文件均为独立完整页面。
- `assets/`：页面引用的公共资源，建议保持路径相对，以确保部署后正常加载。
